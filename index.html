For maximum efficiency, please consider reusing this code, implement the following function in code.js:

Function to be Impl. Explaination
data_location Located the first line of the row; make sure the data is placed in the correct location.
draw_scatterpoints() Finish drawing the scatterpot by adding axis and legends, and append data points based on the loaded data iris_data.
assign_brush_events() Highlight the data based on dragging using D3 brush feature. You should implement the missing 3 sub-functions:
  • brushstarted(): clear any previous selection before brush begins.
  • brushed(): make points outside the selection looks "deselected". You can set the class of those points to deselected, as predefined in this css.
  • brushend(): check if there are no selection, highlight all circles.

Once all functions are correctly implemented, you will get the ready-to-submit result shown above.
